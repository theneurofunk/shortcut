# Команда CryptoAssistant

Эта команда показывает курс BTC в рублях через CoinGecko.

## Как собрать в приложении «Команды»

1. Создай новую команду с названием `Криптокурсы`.

2. Добавь действие `Получить содержимое URL`.
   URL:
   `https://api.coingecko.com/api/v3/simple/price?ids=bitcoin,ethereum&vs_currencies=rub&include_24hr_change=true`
   Метод: `GET`.

3. Добавь действие `Получить значение словаря`.
   В поле ключа укажи:
   `bitcoin`

4. Добавь еще одно действие `Получить значение словаря`.
   В поле ключа укажи:
   `rub`

5. Добавь действие `Задать переменную`.
   Название переменной:
   `btc`

6. Добавь действие `Текст`:

   ```text
   Bitcoin (BTC): btc ₽
   ```

   Вместо слова `btc` вставь переменную из предыдущего действия.

7. Добавь действие `Показать результат`.

8. Нажми кнопку «Поделиться» и выбери `Скопировать ссылку iCloud`.

## Веб-версия

`CryptoAssistant.html` делает то же самое как мини web app. Ее можно открыть на iPhone и добавить на главный экран через «Поделиться» -> «На экран Домой».
